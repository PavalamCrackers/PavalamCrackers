# Pavalam Crackers E-commerce Website

## Overview
Pavalam Crackers is an e-commerce website dedicated to selling a variety of firecrackers. This project aims to provide a user-friendly interface for customers to browse products, learn about the company, and make purchases online.

## Project Structure
```
pavalam-crackers
├── index.html          # Main entry point of the website
├── products.html       # Displays all available products
├── about.html          # Information about Pavalam Crackers
├── contact.html        # Contact form and shop details
├── cart.html           # Shopping cart page
├── css
│   └── style.css       # Styles for the website
├── js
│   └── script.js       # JavaScript functionality
├── assets
│   └── logo.svg        # Logo for the website
├── data
│   └── products.json    # Product data in JSON format
└── README.md           # Project overview and setup instructions
```

## Features
- **Responsive Design**: The website is designed to be fully responsive, ensuring a seamless experience on both desktop and mobile devices.
- **Product Display**: Users can view products with images, descriptions, and prices.
- **Shopping Cart**: Customers can add products to their cart and review their selections before checkout.
- **Contact Form**: A dedicated page for customers to reach out with inquiries.
- **Dynamic Content**: Product information is loaded dynamically from a JSON file.

## Setup Instructions
1. **Clone the Repository**: 
   ```
   git clone <repository-url>
   ```
2. **Navigate to the Project Directory**:
   ```
   cd pavalam-crackers
   ```
3. **Open `index.html`** in your web browser to view the website.

## Technologies Used
- **HTML**: For structuring the web pages.
- **CSS**: For styling the website and ensuring responsiveness.
- **JavaScript**: For adding interactivity and dynamic content loading.
- **JSON**: For managing product data.
